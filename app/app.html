Hello World!

Welcome to multifarce, the text adventure with an infinite number of endings (well, that's the idea anyways)! You can start playing it immediately below, or you can register to be able to continue the story!

It works like this. The game always starts on the The Beginning frame. Now you're able to enter any command you can think of. If the command does not exist, you have the ability to create it (and select/create the frame it will lead to) and continue the story!

To begin your adventure, type start. You can restart the game at any time by typing reset.
